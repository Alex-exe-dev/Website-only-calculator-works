Daten wirklich unwiederruflich löschen?

Nach dieser Aktion können Ihre Daten nicht wiederhergestellt werden.
Wenn Sie dies verstanden haben klicken Sie bitte auf löschen.

Daten Löschen


Auf dieser Seite können Sie alle Daten löschen, die lokal von Ihnen in Ihrem browser gespeichert sind.
Dies sorgt dafür, dass alle eingegebenen Werte unwiederruflich gelöscht werden.
Alles was also gelöscht ist, kann nicht wieder hergestellt werden.
Sollten Sie noch ein Export-File haben, können Sie dieses jederzeit wieder hochladen und die enthaltenen Daten nutzen.
Online werden keine Daten von Ihnen erhoben, dementsprechend kann und wird dort nichts gelöscht werden.
